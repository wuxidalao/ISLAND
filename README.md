#ISLAND#

学习7月老师-微信小程序+服务系统


#介绍#

《island-mini-test》接口测试文件，用微信小程序打开

《island-mini》微信小程序文件

《island-img》项目图片展示

《island》服务器文件

项目使用mysql、node(sequelize/lodash/validate……)、koa(ctx/async/await)、vue


#环境#

mysql:5.6

node:8.0


#启动服务器#

cd island

npm install

npm run dev


#预览项目#

微信开发者工具打开《island-mini》
